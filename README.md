# best-findings
## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Deploy to [GitHub Pages](https://pages.github.com/)

```sh
npm install -g gh-pages
gh-pages -d dist
```
